Módulo 01: Introdução

Clique para saber mais

Módulo introdutório onde explicamos o funcionamento do curso e do suporte.

Módulo 02: Saindo do zero

Clique para saber mais

Módulo pensado especificamente para o aluno que está começando do zero. Aqui abordaremos detalhes de posicionamento dos dedos para montagem dos acordes, diversas posturas necessárias para que os acordes soem limpos, exercícios de digitação, exercícios de mudança de acordes, compassos, a lógica por trás da batida de pagode, etc. É um módulo imprescindível para quem está no início da jornada no cavaquinho.

Módulo 03: Descomplicando a Harmonia

Clique para saber mais

Nesse módulo, reuni os conceitos teóricos que considero imprescindíveis para que você possa tocar cavaco com segurança e desenvoltura. Aqui estudamos escalas, formação de acordes, campos harmônicos, funções harmônicas, modulações e etc. É aqui que você vai aprender como funcionam as músicas. Após esse módulo você será capaz de analisar as músicas e dar um passo importantíssimo para não depender mais de cifras e músicas simplificadas.

Módulo 04: Cadências

Clique para saber mais

Nessa parte do curso, separamos uma aula especial falando sobre as cadências mais utilizadas do samba e pagode. Imprescindível para quem quer tocar músicas de ouvido.

Módulo 05: Entendendo os Shapes Dissonantes

Clique para saber mais

Aqui você vai aprender dicas práticas para executar os acordes dissonantes. Aqueles acordes “complicados” (ex: Am7/9) que os iniciantes correm para simplificar. Nesse módulo você vai entender a lógica por trás da montagem desses acordes no braço do cavaco e não vai mais precisar ficar no decoreba para aprender posições novas no cavaco.

Módulo 06: Dominando as Sequências

Clique para saber mais

Nessa parte do treinamento vamos te apresentar várias sequências de acordes que se repetem em várias músicas. Conhecer essas sequências vai te ajudar a aumentar seu repertório, uma vez que essas sequências aparecem em várias músicas. Um exemplo clássico das sequências é o famoso quadradinho. Aprender as sequências ajudam também a desenvolver o seu ouvido.

Módulo 07: Analisando músicas harmonicamente

Clique para saber mais

Aqui separamos várias músicas para analisarmos. Analisamos as músicas para entender a lógica por trás delas. Entendendo a estrutura de determinada música, você terá condições de expandir esse conhecimento para outras músicas e ficará cada vez mais fácil de você aprender músicas novas e ampliar seu repertório. Assim como as sequências, analisar as músicas também nos ajuda a desenvolver nossos ouvidos.

Quem é Victor Cazzoli:

Praticante de cavaco há 16 anos (e de violão há mais de vinte anos), sou o criador do projeto “Cavaquinho do Zero – por Victor Cazzoli” no qual disponibilizo vídeos no YouTube, Instagram e Facebook ensinando cavaquinho com foco no público que está dando (ou ainda não deu) os primeiros passos no instrumento.
